### For Administrators:

**Managing Students:**
1. Tap "Students" tab
2. Use search bar to find student
3. Or filter by class
4. Tap student card for details
5. Use QR icon to generate code
6. Use edit icon to update info

**Generating Reports:**
1. Tap "Reports" tab

3. View quick stats

1. Tap respective tab (Students/Teachers/Classes)

3. Fill in details
4. Save
